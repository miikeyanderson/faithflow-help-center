# Notification Preferences

Configure when and how Faithflow sends you notifications.

## Notification Types

### Daily Reminders
- Reminds you to practice and maintain your streak
- Set custom reminder time
- Choose how many reminders per day

### Progress Updates
- Level up notifications
- Achievement unlocks
- Streak milestones

### Social Notifications
- Friend requests
- Community activity
- Challenges and events

## Configuring Notifications

### In-App Settings
1. Open **Settings** → **Notifications**
2. Toggle notification types on/off
3. Set reminder times
4. Choose notification style

### iOS System Settings
1. iOS Settings → Faithflow → Notifications
2. Allow Notifications (toggle on)
3. Choose alert style:
   - Lock Screen
   - Notification Center
   - Banners
4. Enable sounds and badges (optional)

## Quiet Hours

Set times when notifications are paused:
1. Settings → Notifications → Quiet Hours
2. Enable Quiet Hours
3. Set start and end time
4. Critical notifications may still appear

## Troubleshooting

**Not receiving notifications?**
1. Check iOS Settings → Faithflow → Notifications are enabled
2. Ensure notifications are enabled in-app
3. Check Do Not Disturb is off
4. Restart the app

**Too many notifications?**
- Adjust frequency in Settings
- Disable specific notification types
- Enable Quiet Hours
