# Common Issues

Quick solutions to frequently encountered problems.

## Login & Account Issues

### Can't Sign In
**Problem**: Unable to log in with Apple ID or email

**Solutions**:
1. Ensure you're using the correct Apple ID or email
2. Check your internet connection
3. Try "Forgot Password" if using email
4. Update to the latest app version
5. Restart the app

### Account Data Not Syncing
**Problem**: Progress not syncing across devices

**Solutions**:
1. Ensure you're signed into the same account
2. Check internet connection
3. Force close and reopen the app
4. Tap "Sync Now" in Settings (if available)

## Lessons & Progress

### Lesson Won't Load
**Problem**: Lesson stuck loading or shows error

**Solutions**:
1. Check internet connection
2. Close and restart the app
3. Clear app cache in Settings
4. Update to latest version

### Progress Not Saving
**Problem**: Completed lessons not showing as complete

**Solutions**:
1. Complete the entire lesson (don't exit early)
2. Ensure internet connection is stable
3. Wait a few seconds for sync to complete
4. Check progress in your profile

### Missing XP or Hearts
**Problem**: XP or hearts not credited after lesson

**Solutions**:
1. Wait a few minutes for sync
2. Pull down to refresh in the app
3. Check if lesson was fully completed
4. [Contact support](./contact-support) if issue persists

## Notifications

### Not Receiving Notifications
**Problem**: No daily reminders or notifications

**Solutions**:
1. iOS Settings → Faithflow → Notifications → Enable
2. In-app: Settings → Notifications → Enable reminders
3. Check Do Not Disturb is off
4. Restart device

## Purchases & Subscriptions

### Purchase Not Working
**Problem**: Can't complete subscription or purchase

**Solutions**:
1. Ensure payment method is valid in App Store
2. Check App Store account is signed in
3. Restart device
4. Try purchase again

### Subscription Not Active
**Problem**: Purchased subscription but don't have premium features

**Solutions**:
1. Settings → Subscription → Restore Purchases
2. Wait a few minutes and restart app
3. Verify purchase in App Store subscriptions
4. [Contact support](./contact-support) with receipt

## Performance Issues

### App Crashes or Freezes
**Problem**: App crashes, freezes, or is slow

**Solutions**:
1. Update to latest app version
2. Restart your device
3. Free up device storage
4. Reinstall the app (progress will sync if signed in)

### Audio/Video Not Playing
**Problem**: Lesson media won't play

**Solutions**:
1. Check device volume and mute switch
2. Ensure internet connection is stable
3. Try different lesson
4. Restart app

## Still Need Help?

If these solutions don't resolve your issue:
1. [Contact Support](./contact-support) with details
2. Include:
   - Device model and iOS version
   - App version
   - Description of the issue
   - Screenshots (if applicable)
